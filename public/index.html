¿Que es Help Desk?

Es una herramienta creada por Cenfotec Software House para brindarle una mejor administración y un mantenimiento adecuado a los proyectos que tengan los clientes con Cenfotec Software House. El principal objetivo de esta aplicación es brindar seguridad y mayor control sobre los datos y proyectos que se trabajan. Además esta aplicación le brinda servicios de mantenimiento por un período determinado, esto con el fin de solucionar errores o realizar cambios que se necesiten.
