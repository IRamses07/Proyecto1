¿Que es Help Desk?

Es una herramienta creada por Cenfotec Software House para brindarle una mejor administración y un mantenimiento adecuado a los proyectos que tengan los clientes con Cenfotec Software House. El principal objetivo de esta aplicación es brindar seguridad y mayor control sobre los datos y proyectos que se trabajan. Además esta aplicación le brinda servicios de mantenimiento por un período determinado, esto con el fin de solucionar errores o realizar cambios que se necesiten.

¿Quienes Somos?

Cenfotec Software House es un proyecto de emprendimiento académico y profesional dentro de la Universidad Cenfotec, en el cual los estudiantes les permite vivir una experiencia de trabajo reales, y con un cliente final, en base a esta propuesta es que nace Cenfotec Software House. Cenfotec Software House brinda a los estudiantes que trabajan en los proyectos, becas como remuneración a su participación.

“Los estudiantes que participan del Cenfotec Software House, son estudiantes que han demostrado un nivel de excelencia mayor, que cuentan con altas habilidades tanto técnicas como blandas y con el deseo de ser parte del equipo.”

-Universidad Cenfotec 2016

“Cada equipo de trabajo del Cenfotec Software House, cumple con las etapas y las mejores prácticas, según cada caso. Además , se mantendrá en periódica comunicación con el interesado para validar diferentes aspectos del proyecto.”

-Universidad Cenfotec 2016

¿Help Desk?

Fácil administración de proyectos en todas sus etapas de diseño, desarrollo y soporte.

El propósito principal de esta aplicación web es brindar un método de comunicación y punto de contacto de fácil acceso para todos nuestros colaboradores y clientes con la finalidad de contar con un sistema para la administración para los proyectos de Cenfotec Software House, además de acompañamiento durante todo el proceso de diseño, implementación, testeo y soporte por el que cualquier software de alta calidad debe pasar.
